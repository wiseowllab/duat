Phase 1 — Codexへの初回指示
DUATという古代エジプト落ちゲーを作ります。
Phaser.jsを使い、以下の構造でファイルを作成してください。

src/main.js — Phaser設定・起動
src/scenes/GameScene.js — メインゲーム
src/core/Board.js — 6×12グリッド管理
src/core/Piece.js — 2個組ピース・4方向回転
src/core/GravitySystem.js — 落下・着地
src/data/constants.js — 定数定義
index.html — エントリーポイント
docs/ — GitHub Pages用にindex.htmlをコピー

ピースは6色の色付きブロックで仮実装。
左右移動・回転・着地・ゲームオーバーまで動く状態にしてください。
Phase 2 — カノポスセット消去の追加
src/core/CanopusResolver.jsを新規作成してください。

仕様：
- 肝臓・肺・胃・腸の4種が1つの隣接グループ内に全種含まれると消去
- 心臓ピースはワイルドとして不足している種を補える
- 脳ピースはセットに参加しない
- 既存のMatchResolver.jsと並走させる
- 消去時は同種消去より高いスコアを返す
棺メーター・神復活システム追加
src/data/gods.jsとsrc/ui/CoffinMeter.jsを作成してください。

gods.jsの仕様：
- Tier1〜4の神データを配列で定義
- 各神に名前・Tier・必要得点・ボム種別を持たせる

CoffinMeter.jsの仕様：
- 棺型の縦ゲージをPhaser上に描画
- 得点に応じてゲージが上昇
- 満タンで神復活イベントを発火する
- 次の棺に自動交代する
